# GoogleSQL.com Phase 1 Plan

## Goal

Acquire developers and data analysts through SEO content and free GoogleSQL
tools.

## Product Surface

- Product-first homepage with the GoogleSQL Copilot workbench.
- Tool routes for Text to GoogleSQL, SQL Explain, and SQL Optimize.
- Tutorial index for GoogleSQL learning content.
- Cheat sheet index for high-intent search traffic.
- Pricing route with Free, Pro, and Team placeholders for later conversion.

## Phase 1 Acceptance Criteria

- A visitor can open the homepage and immediately use the Copilot demo.
- Generated SQL is read-only and includes cost-control guidance.
- Unsafe pasted SQL is detected before execution.
- Tutorials and cheat sheets are crawlable pages.
- CI runs lint, typecheck, unit tests, and production build.
- Health endpoint returns a machine-readable success payload.

## Out of Scope

- Live BigQuery execution.
- User authentication.
- Billing.
- Workspace persistence.
- Enterprise Shield controls.
